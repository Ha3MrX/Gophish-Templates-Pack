## 🚀 How to Use
1. **Landing Pages:** Copy the HTML from the `index.html` file of the platform you want to test.
2. **Email Templates:** Use the `email.html` or `email.txt` files for your campaign messages.
3. **Capture Data:** Ensure you check "Capture Submitted Data" in your Gophish settings.
